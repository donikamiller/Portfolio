Writing Samples

 

 

These samples were developed in XML, HTML, or Markdown, using tools including MadCap Flare, Confluence, Oxygen, and VisualStudio. I also built the searchable help sites where these samples were originally published.

User Guide

This information serves a novice user base by using detailed definitions, parameters, and instructions. It is part of a complete application guide for a SaaS content and data management system, and covers a web utility that augments the primary application.

 

Workflow Guide

This excerpt serves doctors, nurses, office staff, and other medical professionals who engage with an Electronic Medical Records system intermittently, while doing hands-on work with patients. Information is distilled into bite-sized instructions and designed around descriptive headers so that it can be easy to find on-the-fly.

 

User Installation Guide

This guide is for a mixed user base, and describes the process for rebranding the look, feel, and functionality of a webstore. The document is designed to straddle the line between technical/non-technical users in the same way the application itself does.

 

Standard Operating Procedures

This team=specific procedural document includes content from the user guide excerpted on this page. The content is a combination of audience-specific processes, and included or linked content from the primary user guide. This facilitiates rapid development of custom instructions for specific business units, as well as consistency and quick updates across outputs.
